# Developer Summit 2025 Speed Geeking Demo

## Overview
This demo showcases the ease of creating a mapping app using the ArcGIS Maps SDK for JavaScript components and Calcite Design System.
## Resources Used
- [Dev Tech Summit Demo Templates](https://github.com/EsriDevEvents/dev-tech-summit-demo-templates): A collection of demo templates used during the Developer Summit.
- [ArcGIS API for JavaScript](https://developers.arcgis.com/javascript/latest/): Official documentation and resources for the ArcGIS API for JavaScript.
- [ArcGIS API for JavaScript Components](https://developers.arcgis.com/javascript/latest/components/): Detailed information on the components available in the ArcGIS API for JavaScript.
- [Calcite Design System](https://developers.arcgis.com/javascript/latest/components/)
